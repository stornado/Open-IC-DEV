# Open IC DEV

Open IC DEV 是一个基于 `iVerilog`, `SystemC`, `UVM`, `verible`, `verilator`, `oh-my-zsh`，`vscode` 等开源工具链的开发环境。

## 快速开始

```bash
# 克隆项目
git clone https://github.com/stornado/Open-IC-DEV.git

# 进入项目目录
cd Open-IC-DEV

# vscode 打开项目
code .
```
